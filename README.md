## Installation
Clone or Download this Repository
```
git clone https://github.com/hackers-brain/md5-hash-cracker.git
```
Change Directory
```
cd md5-hash-cracker
```
Run the setup.py file
```
python3 setup.py
```
Run the Main File
```
python3 main.py
```
